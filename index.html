How to Use?

Infinity PNG provides 'png' files for you to manually use in your existing projects.

  • Grab the PNG:Get the icon (PNG file) you wanna use for your project. Icons come in PNG and JPG formats, so they'll mesh well with your project.
  • Copy the PNG file into your project folderCopy the PNG file into the folder where you want to add
  • Look for the PNG file:Browse to the location where you downloaded the PNG file and select it.
  • Select:Pick the PNG you need and keep working on your project! :)
